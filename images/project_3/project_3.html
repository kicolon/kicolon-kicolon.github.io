Typography Cook Book

❮ ❯

This project was done for my typography class. I had to create an cook book celebrating some of your favorite types of foods or your favorite snacks. I tried to do a iconic representation by focusing only on type and no images in this piece.

Este proyecto fue hecho para mi clase de tipografía. Tuve que crear un libro de cocina que celebrara algunos de sus tipos de alimentos favoritos o sus bocadillos favoritos. Traté de hacer una representación icónica centrándome solo en el tipo y sin imágenes en esta pieza.
