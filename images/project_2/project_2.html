Typography History

❮ ❯

This project was done for my typography class. I had to create an iconic poster celebrating a font for a the main font families that my professor listed. And I tried to make it into a iconic representation of a abstraction piece.

Este proyecto fue hecho para mi clase de tipografía. Tuve que crear un póster icónico que celebrara una fuente para las principales familias de fuentes que enumeró mi profesor. Intenté convertirlo en una representación icónica de una pieza de abstracción.
